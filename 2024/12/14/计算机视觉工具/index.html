torch

torch 是 PyTorch 的核心库，提供了构建、训练和部署深度学习模型的基础工具。它围绕 张量计算 和 自动微分 构建，同时支持 GPU 加速，具有高效性和灵活性。

以下是 torch 的主要内容和功能模块：

1. 张量（Tensor）

张量是 PyTorch 的核心数据结构，类似于 NumPy 的多维数组，但可以在 GPU 上运行以实现高效计算。

  • 常用操作：

    • 维度操作：view、reshape、squeeze、unsqueeze。
    • 索引和切片：支持 NumPy 风格的索引。
    • 数学运算：如加法、乘法、矩阵乘法（@ 或 torch.matmul）、逐元素运算。
    • 广播机制：支持不同形状的张量自动对齐计算。

2. 自动微分（Autograd）

PyTorch 提供强大的自动微分工具，用于实现反向传播和梯度计算。它通过动态计算图实现，每次前向传播时都会生成新的计算图。

3. 优化工具

虽然 torch.optim 是一个独立模块，但它依赖于 torch 的核心功能。主要包括各种优化器，如 SGD、Adam、RMSprop。

4. 神经网络模块（nn）

torch 的 nn 子模块提供了构建神经网络的工具，主要包括：

  • 层：如全连接层（nn.Linear）、卷积层（nn.Conv2d）、循环层（nn.LSTM）。
  • 激活函数：如 nn.ReLU、nn.Sigmoid。
  • 损失函数：如 nn.CrossEntropyLoss、nn.MSELoss。

5. GPU 加速

torch 支持 GPU 和 CPU 设备之间的灵活切换。

简单总结

torch 是 PyTorch 的核心模块，涵盖了以下功能：

  • 张量计算（Tensor）
  • 自动微分（Autograd）
  • 优化工具
  • 神经网络构建（nn 模块）
  • GPU 加速和分布式计算
  • 数据加载和保存

这些特性让 torch 成为构建深度学习模型的高效工具。 ### torchvision torchvision 是 PyTorch 的一个官方库，专为 计算机视觉任务 提供支持。它包含了处理图像数据的常用工具、预训练模型以及一些流行的数据集，极大地简化了计算机视觉模型的开发和实验过程。

主要模块和功能

  1. torchvision.transforms

    • 图像变换工具，用于数据预处理和增强。常用于对输入图像进行裁剪、缩放、归一化等操作。

    • 常见操作：

      • Resize：调整图像大小。
      • CenterCrop / RandomCrop：裁剪图像。
      • ToTensor：将图像转换为 PyTorch 的张量格式。
      • Normalize：对图像数据进行标准化处理。
      • RandomHorizontalFlip：随机水平翻转。

    • 示例：

  2. torchvision.datasets

    • 提供了常用的 视觉数据集，用户可以直接加载这些数据集，而无需手动下载或格式化。

    • 常见数据集：

      • CIFAR10、CIFAR100
      • MNIST
      • ImageNet
      • COCO（包括目标检测和分割任务）
      • VOC（PASCAL VOC 数据集）

    • 示例：

  3. torchvision.models

    • 提供了大量预训练的 深度学习模型，例如 ResNet、VGG、MobileNet、EfficientNet 等。用户可以直接使用这些模型进行迁移学习或微调。

    • 支持分类、目标检测、实例分割、关键点检测等任务的模型。

    • 示例：

  4. torchvision.io

    • 提供了基础的 图像和视频读写工具。

    • 示例：

  5. torchvision.ops

    • 提供了一些高级的 操作和函数，常用于目标检测和分割任务，例如：

      • nms（非极大值抑制）。
      • roi_align（区域对齐操作）。
      • box_convert（边界框格式转换）。

    • 示例：

应用场景

  • 数据预处理：使用 transforms 对图像数据进行增强或规范化处理。
  • 加载数据集：直接加载和使用流行数据集，便于快速实验。
  • 迁移学习：通过加载预训练模型，加速模型开发。
  • 目标检测和分割：内置支持复杂的视觉任务。
